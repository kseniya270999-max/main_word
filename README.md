## !! ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ !!

Все скрипты сборки и упаковки должны запускаться **ТОЛЬКО из корня проекта**.

Корень проекта — это каталог, содержащий папки:
- src/
- cicd/
- usr/

Запуск скриптов из подкаталога `cicd/` приведёт к ошибкам, связанным с относительными путями.
